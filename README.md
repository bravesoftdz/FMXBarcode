| Linear          | EAN-8       | :white_check_mark: |
|                 | EAN-13      | :white_check_mark: |
|                 | UPC-A       | :white_check_mark: |
|                 | UPC-E       | :white_check_mark: |
|                 | ITF-14      | :white_check_mark: |
|                 | GS-128      | :white_check_mark: |
|                 | Databar     | :x: (future)       |
| Two dimensioanl | DataMatrix  | :x: (future)       |
|                 | QR Code     | :x: (future)       |
|                 | DotCode     | :x: (future)       |
| Composite       |             | :x: (future)       |
